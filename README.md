# Text Analyzer (Python)

A Python-based tool for analyzing text files and generating statistics such as word count, sentence count, and word frequency.

## Features
- Counts total words and sentences
- Calculates words per sentence ratio
- Counts total characters
- Counts characters excluding punctuation
- Finds the shortest and longest words
- Computes word frequency distribution

## Technologies
- Python 3
- Standard Libraries (sys)

## Usage
Run the script with input and output files:

```bash
python text_analyzer.py sample_input.txt sample_output.txt
